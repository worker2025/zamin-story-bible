# Scene Index (Shot/Scene Numbering Reference)

## Numbering Convention
Each Story Bible Event ID (E01-E77) is assigned one Scene Number (SC-001 to
SC-077) in the same order, with its Chapter and Location cross-referenced.
Where a single event covers multiple beats (e.g., dialogue plus action),
no sub-shot breakdown is invented; only the single event boundary recorded
in the Story Bible is used. Production teams requiring finer shot
breakdowns must derive them from the full manuscript text, which is
explicitly out of scope for this phase.
